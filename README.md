# Sonos Controller

> Control your Sonos Speakers and Play Songs from URLs

## Features

* List of all Sonos Devices
* Ability to Control Speakers / Zones
* Can Play from URLs (MP3, WAV, etc.)

## Developmen

### Auth

Default Creds:

* user
* 1234

## Run

With the following command:

```sh
python main.py
```

* NiceGUI
